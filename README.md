# DOCKER NUXT / LUMEN / MYSQL Memory game !!!

## Build Setup

This project require DOCKER !

``` bash

# install project
$ yarn dev:build

# serve project after install
$ yarn dev:up

# down all container
$ yarn dev:down

# down containers and killall
$ yarn dev:killall



```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
